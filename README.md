## Getting started
1. Setup the frontend.

```
cd client
npm install         # download dependencies
npm run dev         # start the frontend server
```

2. Setup the backend.
```
cd server
brew install uv     # you probably need to install uv
uv run fastapi dev  # start the backend server
```
